JavaScript Operators Explained with Examples

Arithmetic Operators

Perform mathematical operations like addition, subtraction, multiplication, division, and exponentiation.

let a = 10, b = 3;
console.log(a + b);  // 13
console.log(a - b);  // 7
console.log(a * b);  // 30
console.log(a / b);  // 3.3333...
console.log(a ** b); // 1000 (10 to the power 3)

Remainder Operator (%)

Returns the remainder of division of one number by another.

let a = 10, b = 3;
console.log(a % b);  // 1

Assignment Operators

Assign values to variables with optional arithmetic operations.

let x = 5;
x += 3;  // x = 8
x -= 2;  // x = 6
x *= 4;  // x = 24
x /= 6;  // x = 4

Unary Operators

Operate on a single operand, like increment (++), decrement (--), negation (-), and logical NOT (!).

let num = 5;
console.log(++num);  // 6 (pre-increment)
console.log(num--);  // 6 (post-decrement, prints then decrements)
console.log(-num);   // -5 (negation)
console.log(!false); // true (logical NOT)

Comparison Operators

Compare two values and return a boolean result.

console.log(5 == '5'); // true (loose equality)
console.log(5 === '5'); // false (strict equality)
console.log(5 != 3); // true
console.log(5 !== '5'); // true
console.log(3 > 2);  // true
console.log(3 <= 3); // true

Logical Operators

Combine boolean values.

console.log(true && false); // false (AND)
console.log(true || false); // true (OR)
console.log(!true);         // false (NOT)
